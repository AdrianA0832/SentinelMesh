# Privacy Guarantee

SentinelMesh is built for "Paranoid-First" users.

## 1. Local Only
- 0% Cloud.
- 0% Telemetry.
- 0% analytics.
- Your data never leaves `localhost` unless *you* configure an SMTP server.

## 2. Metadata Only
- We check *counts*, not *content*.
- We check *who* you talk to, not *what* you say.
- We check *timestamps*, not *logs*.

## 3. Data Ownership
- All data is stored in `data/`.
- You can delete it anytime.
- You can export it via `/export` API.
- Files are plain text/JSON/CSV for easy inspection.
